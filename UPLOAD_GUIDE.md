# 📋 Guide d'Upload Manuel sur GitHub

## 🎯 Objectif
Uploader les fichiers du dossier `docs` sur votre dépôt GitHub pour activer le site

## 📁 Fichiers à uploader (dans l'ordre)

### 1. Allez sur votre dépôt
**URL:** https://github.com/Hassen100/game-magasin

### 2. Créez le dossier `docs`
1. Cliquez sur "Add file" → "Create new file"
2. Nom du fichier: `docs/.gitkeep`
3. Commit: "Create docs folder"
4. Cliquez sur "Commit new file"

### 3. Uploadez les fichiers principaux

#### Fichier 1: index.html
1. Cliquez sur "Add file" → "Upload files"
2. Glissez-déposez le fichier: `index.html`
3. Assurez-vous qu'il va dans le dossier `docs/`
4. Commit: "Add index.html"

#### Fichier 2: main-B6I3CKHY.js
1. Cliquez sur "Add file" → "Upload files"
2. Glissez-déposez le fichier: `main-B6I3CKHY.js`
3. Assurez-vous qu'il va dans le dossier `docs/`
4. Commit: "Add main JavaScript file"

#### Fichier 3: styles-2DHDKEKI.css
1. Cliquez sur "Add file" → "Upload files"
2. Glissez-déposez le fichier: `styles-2DHDKEKI.css`
3. Assurez-vous qu'il va dans le dossier `docs/`
4. Commit: "Add CSS styles"

#### Fichier 4: favicon.ico
1. Cliquez sur "Add file" → "Upload files"
2. Glissez-déposez le fichier: `favicon.ico`
3. Assurez-vous qu'il va dans le dossier `docs/`
4. Commit: "Add favicon"

#### Fichier 5: .nojekyll
1. Cliquez sur "Add file" → "Upload files"
2. Glissez-déposez le fichier: `.nojekyll`
3. Assurez-vous qu'il va dans le dossier `docs/`
4. Commit: "Add .nojekyll file"

### 4. Configuration GitHub Pages
1. Allez sur: https://github.com/Hassen100/game-magasin/settings/pages
2. Source: "Deploy from a branch"
3. Branch: "main"
4. Folder: "docs"
5. Cliquez sur "Save"

### 5. Test final
Attendez 2-3 minutes puis visitez:
https://hassen100.github.io/game-magasin/
